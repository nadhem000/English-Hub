I. 1. Introduction to the Third Conditional

📘 What is the Third Conditional?

📘 Definition:

The third conditional is used to talk about unreal situations in the past. It expresses that the condition did not happen, so the result did not happen either. It often expresses regrets or criticism.

💡 Core ideas:
  • Unreal past: If I had studied, I would have passed the exam.
  • Regrets: If I hadn't been so rude, she wouldn't have left.
  • Hypothetical past results: They would have won if they had trained more.
If I had seen him, I would have told him.
She would have been on time if she had taken the train.
If we had known about the party, we would have gone.
He wouldn't have broken the window if he had been careful.

I. 2. Basic Structure

🔧 Form: If + Past Perfect, Would Have + Past Participle

📘 Rule:

The if-clause uses the past perfect (had + past participle). The main clause uses would have + past participle. The order can be reversed; a comma is used when the if-clause comes first.

StructureExample
If + past perfect,
+ would have + p.p.		 If I had known, I would have helped.
Would have + p.p.
+ if + past perfect		 I would have helped if I had known.
🗣️ Contractions:

In spoken English, "had" is often contracted to 'd, and "would have" to "would've". Example: If I'd known, I'd've called. (If I had known, I would have called.)

❌ Negative Forms:

Use "hadn't" in the if-clause and "wouldn't have" in the main clause.

If you hadn't missed the bus, you wouldn't have been late.
She wouldn't have been upset if you had apologized.

I. 3. Common Uses: Unreal Past & Regrets

💬 Regrets, Criticism & Hypothetical Past Results

😔 Regrets & Criticism:

We use the third conditional to express regret about past actions that cannot be changed, or to criticize past behaviour.

If I had saved more money, I would have bought the car. (I didn't save, I didn't buy)
If they had listened to me, they wouldn't have made that mistake.
🔮 Hypothetical Past Results:

It also describes imaginary alternative past outcomes.

What would you have done if you had seen the accident?
If we had left earlier, we would have caught the plane.
