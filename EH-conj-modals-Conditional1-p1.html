I. 1. Introduction to the First Conditional

📘 What is the First Conditional?

📘 Definition:

The first conditional is used to talk about real and possible situations in the future. If the condition is met, the result is likely to happen. It expresses a cause‑and‑effect relationship.

💡 Core ideas:
  • Real future possibilities: If it rains tomorrow, we will cancel the picnic.
  • Promises and warnings: If you study hard, you will pass the exam.
  • Using “unless” meaning “if not”: Unless you hurry, you will miss the train.
If she calls me, I will tell her the news.
If you don’t sleep enough, you will feel tired.
Unless we leave now, we will be late.
He will get the job if he prepares well.

I. 2. Basic Structure

🔧 Form: If + Present Simple, Will + Infinitive

📘 Rule:

The if‑clause uses the present simple tense; the main clause uses will + base verb. The order can be reversed without changing meaning. A comma is used when the if‑clause comes first.

StructureExample
If + present simple,
+ will + infinitive		If you study, you will succeed.
Will + infinitive
+ if + present simple		You will succeed if you study.
🔁 Using “unless”:

“Unless” means “if not”. It replaces a negative if‑clause. Structure: Unless + present simple, will + infinitive.

Unless you apologize, she will not forgive you. (= If you do not apologize...)
We will miss the flight unless we hurry.
❌ Negative Forms:

Use don’t / doesn’t in the if‑clause and won’t in the result clause.

If you don’t rest, you will get sick.
She won’t pass unless she studies harder.
