Second Conditional: Introduction & Basic Form

Learn how to talk about unreal, improbable or imaginary situations using if + past simple, would + infinitive.

Introduction to Second Conditional

✨ What is the Second Conditional?

Definition:

The second conditional is used to talk about unreal, imaginary or improbable situations in the present or future. It expresses that the condition is not true or is very unlikely to happen.

Core ideas:
  • Imaginary present/future: If I won the lottery, I would buy a house.
  • Unreal situations: If I were you, I would apologize. (were for all persons)
  • Hypothetical advice: If she studied more, she would get better grades.
Examples:
If I had a car, I would drive to work.
If we lived in Paris, we would speak French.
He would travel more if he had more money.
What would you do if you saw a ghost?

Basic Structure

📐 Form: If + Past Simple, Would + Infinitive

Rule:

The if-clause uses the past simple tense (or 'were' for all persons). The main clause uses would + base verb. The order can be reversed; a comma is used when the if-clause comes first.

StructureExample
If + past simple, + would + infinitiveIf I knew her number, I would call her.
Would + infinitive + if + past simpleI would call her if I knew her number.
Using "were" for all subjects:

In formal English, 'were' is used instead of 'was' in the if-clause for all persons (especially after "If I were you").

If I were rich, I would buy a yacht.
If she were here, she would help us.
Negative Forms:

Use "didn't" in the if-clause and "wouldn't" in the main clause.

If I didn't have a job, I would travel.
She wouldn't be upset if you apologized.

Common Uses: Unreal & Imaginary Situations

💭 Hypothetical Present / Future

We use the second conditional to imagine a different reality – something that is not true now or is unlikely to happen.

If I spoke Italian, I would live in Rome.
If they offered me the job, I would accept it.

🎭 Advice (If I were you...)

Giving advice with "If I were you" is a classic second conditional structure.

If I were you, I would study more.
If I were him, I wouldn't say that.
