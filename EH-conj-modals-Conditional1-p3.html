Zero Conditional vs First Conditional

⚖️ Key differences

📘 Zero Conditional (general truths)

Structure: If + present simple, present simple. Used for facts, habits, and things that are always true.

If you heat ice, it melts.
If I am tired, I go to bed early.
🔮 First Conditional (real possibility)

Structure: If + present simple, will + infinitive. Used for possible future events and their likely results.

If it rains tomorrow, we will stay home.
If she studies, she will pass the exam.

Common Mistakes with First Conditional

⚠️ Frequent errors & how to fix them

❌ Using 'will' in the if‑clause

Incorrect: If it will rain, we will cancel the picnic. ✅ Correct: If it rains, we will cancel the picnic. (Use present simple after 'if'.)

❌ Mixing zero & first conditional wrongly

Zero = always true; First = specific future. Compare: If you freeze water, it becomes ice (zero) vs If you freeze this bottle, it will crack (first).

❌ Using modals incorrectly in result clause

In first conditional we often use 'will', but can also use 'can', 'may', 'should' for possibility/advice. Example: If you finish early, you can leave.

Real‑Life Context: Travel Agency Role‑Play

🎭 Using First Conditional in a travel dialogue

Conversation: Booking a trip

Travel agent: "If you book before Friday, you will get a 20% discount."
Customer: "Great! If I pay now, can I reserve the seats?"
Agent: "Yes, if you provide your details, I will confirm immediately."
Customer: "What if I need to change the date?"
Agent: "If you change the date more than 7 days ahead, there is no extra fee."
Customer: "Perfect. I'll book now."

Notice: present simple after if, will/can in the main clause. No 'will' inside the if‑part.
