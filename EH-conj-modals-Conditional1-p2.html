II. 1. Modals in the Result Clause

🔧 Using can, may, should in the result

📘 Modals beyond 'will':

In the first conditional, the result clause can include modals like can (permission/ability), may (possibility), or should (advice). The if‑clause remains in present simple.

If you finish your work, you can leave early. (permission)
If you exercise regularly, you may lose weight. (possibility)
If you feel unwell, you should see a doctor. (advice)

II. 2. Imperative in the Result Clause

📋 Giving direct instructions based on conditions

📘 Structure: If + present simple, (then) + imperative

The imperative gives a direct instruction or warning based on a future condition. This structure is common in manuals, advice, and urgent situations.

If you see smoke, leave the building immediately.
If the machine stops, press the reset button.
If you need help, ask the instructor.

II. 3. Other Conjunctions: As Soon As, When

⏱️ Using 'as soon as' and 'when' in first conditional

📘 Time-bound conditions:

Conjunctions like when and as soon as can replace "if" when the condition is certain or time‑bound. They are followed by present simple, while the main clause uses will + infinitive.

As soon as you arrive, I will call you. (immediate sequence)
When she finishes her degree, she will apply for a job. (certain future)
We will start the meeting as soon as everyone is ready.
